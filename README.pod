=head1 NAME

AI::ParaX::Population::BitVector - Create a population of BitVectors for AI::Genetic::Parallel

=head1 VERSION

Version 0.01

=head1 SYNOPSIS

Create a population random BitVectors as a subclass of AI::Genetic::Parallel::Population

    use AI::ParaX::Population::BitVector;

    $pop => AI::ParaX::Population::BitVector->new(
        individual_count => 100,
        bits             => 100,
    )

=head1 METHODS

=head2 new

=over 4

=item bits

number of bits per individual
